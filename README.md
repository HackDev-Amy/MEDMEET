# 🩺 MEDMEET
**A Doctor Consultation Booking & Pharmacy WebApp**

MEDMEET is a comprehensive healthcare web platform designed to simplify the process of connecting patients with doctors. The system allows users to seamlessly book medical appointments, purchase medicines, and even schedule appointments with international hospitals.

## 🚀 Project Overview
MEDMEET aims to provide a digital healthcare solution where patients can easily find doctors, schedule appointments, and interact with medical professionals online. The platform helps reduce waiting times in hospitals and improves overall access to healthcare services.

## ✨ Features
* **👨‍⚕️ User Roles:** Dedicated registration and dashboards for Doctors and Patients.
* **📅 Smart Scheduling:** Online appointment booking system.
* **🔍 Advanced Search:** Find doctors easily by their specialization.
* **📱 Responsive Design:** A user interface that works beautifully on desktop and mobile.
* **🔐 Security:** Secure authentication and authorization system.

## 🛠️ Tech Stack
* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python, Flask 
* **Database:** MySQL
* **Version Control:** Git & GitHub
